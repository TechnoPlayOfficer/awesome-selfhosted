# ACCET Oracle — README

## Legal Oracle Platform for Tokenized Assets

> **Vision:** Every diploma and real-world asset earns a "Legal Armor" — immutable, state-certified, globally verifiable.

### Stack
- **Frontend:** Next.js + Glassmorphism (`#020624` / `#5AC4EE`)
- **E-Signature:** Docuseal + Documenso + OpenSign
- **Storage:** Filecoin via Lighthouse SDK
- **Blockchain:** Base L2 (SBT/ERC-3643) + Avalanche C-Chain
- **Backend:** Node.js/Express Oracle API
- **ERP:** Odoo + Formance Ledger
- **Automation:** n8n

### Quick Start
```bash
git clone https://github.com/TechnoPlayOfficer/accet-oracle
cd accet-oracle
cp .env.example .env
docker-compose up
```
